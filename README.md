## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd adebayo-entro-test
```

Install dependencies

```bash
  npm install
  npx prisma migrate dev --name addData
```

Start the server

```bash
  npm run dev
```
